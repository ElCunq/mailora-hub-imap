# Gmail Test Results (Archived)

## Summary
- **IMAP:** Works stable on `imap.gmail.com:993` with SSL/TLS.
- **SMTP:** sending works on `smtp.gmail.com:587` with STARTTLS.
- **Authentication:** App Passwords are required (2FA enabled).
- **UID Behavior:** Gmail assigns UIDs asynchronously for Sent items (see `KNOWN_ISSUES.md`).
- **Labels:** Mapped as folders. `[Gmail]/All Mail`, `[Gmail]/Sent Mail`, etc.

*This file is kept for historical reference. See `TEST_RESULTS.md` for current status.*
